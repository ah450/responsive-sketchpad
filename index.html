Initialize Sketchpad

var sketchpad = $('canvas-selector').sketchpad({
    aspectRatio: 2 / 1,             // (Required) To preserve the drawing, a aspect ratio must be specified
    backgroundColor: '#ffffff'      // (Optional) To set the background of the canvas
    strokes: 'JSON',                // (Optional) To initialize the sketchpad with stroke data
});

Color

Select the line color

sketchpad.setLineColor('#000000')
sketchpad.getLineColor()
Random Color

Size

Select the line Size

sketchpad.setLineSize(3)
sketchpad.getLineSize()
Random Size

Undo/Redo

Undo or redo lines drawn

sketchpad.undo()
sketchpad.redo()
Undo Redo

Clear

Reset the canvas

sketchpad.clear()
Clear

Get JSON

Get the canvas strokes in a JSON format

sketchpad.json()

Load JSON

Display JSON array of strokes

sketchpad.jsonLoad('JSON')
